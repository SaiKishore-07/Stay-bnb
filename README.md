# Airbnb Clone

A responsive Airbnb Clone website built using HTML and CSS.

## Features

* Responsive Design
* Navigation Bar
* Search Section
* Exclusive Destinations
* Trending Places
* Traveller Stories
* Listing Page
* House Details Page
* Mobile-Friendly Layout

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
* Font Awesome

## Project Pages

* Home Page
* Listing Page
* House Details Page

* Semantic HTML

* CSS Flexbox
* CSS Grid
* Media Queries
